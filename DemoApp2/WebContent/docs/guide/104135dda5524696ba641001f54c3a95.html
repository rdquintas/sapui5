File Names and Encoding

Specifies the rules for file names and encoding

Some target platforms of SAPUI5 impose technical restrictions on the naming or structure of resources (files). Hence, the following rules for file names and encoding apply:

  • Folder names must not contain spaces (SAP HANA).

  • To avoid issues with the SAPUI5 module loading and with URL handling in general, resource names should not contain spaces.

  • Single folder names must not be longer than 40 characters (ABAP).

  • Two resource names must not only differ in case (ABAP, some J2EE servers).

  • Avoid non-ASCII characters in resource names.
